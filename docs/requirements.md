# Requisitos Funcionais

Funcionalidades esperadas do sistema Finanpy.

## Autenticação

- [x] Cadastro de usuário com Nome, Email e Senha.
- [x] Login via Email e Senha.
- [x] Logout.
- [x] Proteção de rotas (apenas usuários autenticados acessam o sistema).

## Gestão de Contas

- [x] Criar, Editar e Excluir contas bancárias.
- [x] Listar contas com seus saldos atuais.
- [x] Suporte a diferentes tipos (Corrente, Poupança, Carteira, etc).

## Gestão de Categorias

- [x] Criar categorias personalizadas.
- [x] Definir tipo (Receita ou Despesa) e cor.

## Transações

- [x] Registrar entradas (Receitas) e saídas (Despesas).
- [x] Vincular transação a uma conta e categoria.
- [x] Listagem histórica de transações.

## Dashboard

- [x] Visão geral do saldo total.
- [x] Resumo de receitas e despesas do mês.
- [x] Gráficos ou listas simplificadas para acompanhamento rápido.

## Perfil

- [x] Visualizar e editar dados do perfil (Avatar, Bio, Telefone).
