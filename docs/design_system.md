# Design System

Guia de estilos e componentes visuais do Finanpy.

## Cores

A paleta principal baseia-se em tons de **Cyan** e **Emerald** sobre fundo escuro (Dark Mode).

| Cor | Uso | Exemplo |
|-----|-----|---------|
| **Cyan-500** | Ações Primárias, Links | `#06b6d4` |
| **Emerald-500** | Sucesso, Receitas | `#10b981` |
| **Red-500** | Erro, Despesas, Perigo | `#ef4444` |
| **Gray-950** | Background Principal | `#030712` |
| **Gray-900** | Superfícies (Cards) | `#111827` |

## Tipografia

- **Fonte Principal**: Sans-serif (Inter ou System UI).
- **Títulos**: Texto claro (`gray-100`), pesos Bold ou Semibold.
- **Corpo**: Texto `gray-300` para boa legibilidade em fundo escuro.

## Componentes Comuns

### Botões

- **Primário**: Fundo Cyan, Texto Branco. Hover levemente mais escuro.
- **Secundário**: Fundo Gray-700, Texto Gray-100.
- **Danger**: Fundo Red-500, para ações destrutivas.

### Formulários

- Inputs com fundo `gray-800` e borda `gray-700`.
- Foco com anel (ring) na cor `cyan-500`.

### Cards

- Fundo `gray-900` com borda sutil `gray-800`.
- Cantos arredondados (`rounded-xl`).
- Gradientes sutis podem ser usados para destaque.
