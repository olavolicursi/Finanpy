# Finanpy Documentation

Bem-vindo à documentação do **Finanpy**, um Sistema de Gestão de Finanças Pessoais.

Esta documentação tem como objetivo guiar desenvolvedores e usuários sobre o funcionamento, arquitetura e padrões do projeto.

## Índice

- [Instalação e Configuração](setup.md) - Guia passo a passo para rodar o projeto localmente.
- [Arquitetura](architecture.md) - Visão geral da stack tecnológica e estrutura de diretórios.
- [Banco de Dados](database.md) - Modelagem de dados e dicionário de dados.
- [Design System](design_system.md) - Guia de estilos, cores e componentes UI.
- [Requisitos Funcionais](requirements.md) - Lista de funcionalidades e regras de negócio.

## Sobre o Projeto

O **Finanpy** é uma aplicação web desenvolvida com Django e TailwindCSS para controle financeiro pessoal, permitindo gestão de contas, transações e visualização de despesas.
